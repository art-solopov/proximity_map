== README

Система нахождения ближайших точек.

Автор: Солопов А. И.

Использование: кликнуть на карту. Получить таблицу с точками в радиусе 4 км.

Карта и список зданий Москвы получены из OpenStreetMap: http://www.openstreetmap.org

Виджет карты отображается посредством OpenLayers: http://openlayers.org
